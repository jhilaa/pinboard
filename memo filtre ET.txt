
Pour mettre en place un filtrage de type "et" en utilisant des cases à cocher pour filtrer les éléments en fonction de leurs tags, vous pouvez suivre ces étapes en JavaScript :

Ajoutez un gestionnaire d'événements aux cases à cocher pour détecter les changements d'état.

Lorsqu'une case à cocher est cochée ou décochée, parcourez tous les éléments .pin et vérifiez s'ils correspondent aux critères sélectionnés.

Pour chaque élément .pin, vérifiez s'il possède tous les tags correspondant aux cases à cocher cochées. Si c'est le cas, affichez cet élément ; sinon, masquez-le.

Voici le code JavaScript correspondant :

document.addEventListener("DOMContentLoaded", function () {
    // Sélectionnez toutes les cases à cocher
    const checkboxes = document.querySelectorAll("#checkboxes_container input[type=checkbox]");

    // Ajoutez un gestionnaire d'événements pour chaque case à cocher
    checkboxes.forEach(function (checkbox) {
        checkbox.addEventListener("change", filterPins);
    });

    // Fonction pour filtrer les éléments .pin
    function filterPins() {
        // Récupérez tous les éléments .pin
        const pins = document.querySelectorAll(".pin");

        // Parcourez les cases à cocher pour obtenir les critères sélectionnés
        const selectedCriteria = [];
        checkboxes.forEach(function (checkbox) {
            if (checkbox.checked) {
                selectedCriteria.push(checkbox.name);
            }
        });

        // Parcourez les éléments .pin et vérifiez s'ils correspondent aux critères sélectionnés
        pins.forEach(function (pin) {
            const tags = pin.querySelectorAll(".tag");
            let shouldShow = true;

            // Parcourez les tags de l'élément .pin
            tags.forEach(function (tag) {
                const tagText = tag.textContent.trim();
                // Vérifiez si le tag n'est pas présent dans les critères sélectionnés
                if (!selectedCriteria.includes(tagText)) {
                    shouldShow = false;
                }
            });

            // Affichez ou masquez l'élément .pin en fonction du résultat
            if (shouldShow) {
                pin.style.display = "block";
            } else {
                pin.style.display = "none";
            }
        });
    }
});
Ce code JavaScript ajoute un gestionnaire d'événements aux cases à cocher pour filtrer les éléments .pin en fonction des critères sélectionnés. Il parcourt les tags de chaque élément .pin et les compare aux critères sélectionnés pour décider s'il faut afficher ou masquer l'élément.
